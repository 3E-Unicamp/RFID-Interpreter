------- [3E] RFID Interpreter -------
A finalidade deste projeto é aprender a utilizar as ferramentas básicas 
de programação para criar um dispositivo, utilizando o ESP32 ou ArduinoUNO,
que possa detectar o RFID associado a cada pessoa que entra na sede para os plantões.

Para a elaboração deste projeto precisamos de:
- Conhecimento de C / C++
- Conhecimento de GitHub para a administração de versões de código
